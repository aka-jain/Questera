# Questera
A beautifully designed Questionnaire based on API made with Angular

#API Response Format
```json
[
	{
		"quest":"What's your name?",
		"options":[
			{
				"option": "Name",
				"selected": false,
				"value":"a"
			},
			{
				"option": "Raun",
				"selected": false,
				"value":"b"
			}

		],
		"ans":"a"
	}
]
```
